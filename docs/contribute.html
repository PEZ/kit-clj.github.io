How you can help

Kit is an open source project and there are many ways to get involved.

Report Bugs

The project is hosted on GitHub. If you have feature requests or would like to submit improvements, don't hesitate to open an issue or submit a pull request.

Help Improve Documentation

The source for this site along with the associated documentation is hosted on GitHub.

If you wish to update an existing documentation page, simply edit the Markdown file for the page here.

To add a new topic simply create a new Markdown file under resources/md directory, and update the list of topics here.
